📸 頭像圖片替換說明
===================

🔧 如何替換頭像：
1. 將您的照片檔案重新命名為 "profile.jpg"
2. 將檔案放入這個資料夾 (src/assets/images/)
3. 重新整理網頁即可看到新的頭像

💡 建議的圖片規格：
• 檔案格式：JPG, PNG, WebP
• 圖片尺寸：建議 400x400 像素或以上的正方形
• 檔案大小：建議 200KB 以下（確保載入速度）
• 圖片品質：清晰、光線充足的正面照
• 背景：建議使用簡潔或專業的背景

🎨 圖片建議：
• 穿著正式或商務休閒服裝
• 保持自然且專業的表情
• 避免過度修圖
• 確保臉部清楚可見

📋 支援的檔案名稱：
• profile.jpg (主要)
• profile.png
• profile.webp

⚠️ 注意事項：
• 請確保您有該照片的使用權
• 建議先備份原始檔案
• 如果圖片無法載入，系統會顯示預設頭像

🔄 如果圖片沒有顯示：
1. 檢查檔案名稱是否正確
2. 確認檔案格式是否支援
3. 重新整理瀏覽器頁面
4. 檢查檔案是否已上傳到正確位置
